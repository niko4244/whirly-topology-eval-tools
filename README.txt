# Brand Tech Sheet Portal

## Installation
1. Download and run `BrandTechSheetPortalSetup.exe`
2. Follow the installer prompts
3. App launches automatically (browser opens to dashboard)

## Features
- Brand-filtered tech sheet upload and gamification
- Secure admin dashboard and CSV report downloads
- Technician achievements and feedback
- Automatic updates (optional)

## Uninstall
- Use Control Panel > Programs > Uninstall "Brand Tech Sheet Portal"
